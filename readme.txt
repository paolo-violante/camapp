*
* CAMDROID - app android per il repurposing di webcam di computer portatili 
* come videocamera di sorveglianza
*
* ISTRUZIONI D'USO
* 
*  Il progetto gira su server test locali
*
* - startare mongodb all'indirizzo di default mongodb://localhost:27017/
* 
* - startare server flask da terminale
*      flask --app app run --host=0.0.0.0
*
* - copiare la cartella "template" nella cartella dell'app flask
*
* - connettersi a 127.0.0.1:5000/login.html tramite browser e fare login/registrazione
*
* - dopo aver installato l'app inserire username/password (e ip/porta server se necessario) 
*   nella stessa
*   (link app https://www.mediafire.com/file/ctx6r56d6ijh9zw/camapp-1.0-arm64-v8a_armeabi-v7a-debug.apk/file)
*
